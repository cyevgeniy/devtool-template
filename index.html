How do I use this template?



First, clone the template to your account. Then, decide which blocks make sense for your landing and delete the rest. Replace all "Lorem ipsum" text with your copy, upload your images, and set up actions for the buttons and links.

How do I adjust the color theme to match my brand identity?



You can update the color theme in two ways:

  1. Switching between light and dark modes.
    Apply either the Theme Light or Theme Dark class to the Body element of any page. Just select the Body, go to the Style panel on the right, and choose the class you want. This will apply the theme across the entire page. If no class is set, the light theme is used by default.
  2. Customize the color palette.
    Use this tool to tweak the overall color theme. Adjust the two sliders until both themes match your brand's vibe. Then, in the Webflow template, go to Variables → Colors and copy-paste the color codes from the tool into the template's variables.

How do I edit text?



In Designer mode, simply double-click any text block and copy-paste your content.

How do I change images?



Open the Assets panel. Hover over an image, click the "…" icon, and select Replace to update all instances of that image. To upload a new one, click the Upload icon, then drag and drop it into the desired spot in your layout's hierarchy.

How can I use a different icon?



This template uses Font Awesome icons as font files, which makes it easy to add them to buttons and links and style them like regular text. Brand icons, such as logos, are stored separately. To use them, apply the Icon Brand M class to any text element — this enables Font Awesome Brand icons. For standard icons, use classes from Icon M to Icon XL depending on the size you need.

To add a new icon:

  1. Go to fontawesome.com/icons
  2. Search for an icon in the free set
  3. Copy the icon as text
  4. In Webflow, paste it into a text block that has any of the 'Icon' classes applied. Pay attention to the icon style — some icons are available only in the Solid style

How do I add a new page?



First, you'll need to upgrade your Webflow site plan. The free Starter plan used in this template only allows up to 2 pages.

Once upgraded, click the Pages icon in the top-left corner. When the Pages panel opens, scroll down to the Templates section, find "Content Page template," hover over it, and click the plus (+) icon next to it.

The new page will include all available elements — just delete anything you don't need.

Where can I see all the pre-designed elements this template includes?



Open the Pages panel by clicking the icon in the top-left corner. Then, scroll down to the Templates section and click on "Content Page template." This page includes all the pre-designed elements, ready for you to copy-paste into your own pages.

How do I publish my website?



There are two ways to bring your site to life:

  1. Publish it using Webflow's hosting
    In Webflow, click Publish in the top-right corner and follow the instructions in this article. This works with any plan, including the free one. However, you'll need to upgrade if you want to integrate third-party analytics or use the built-in Webflow analytics plugin
  2. Publish it using your own hosting
    To do this, you need to purchase any paid plan to access the export code feature. Once you download the code, you can host the website in any convenient way and cancel your subscription afterward
